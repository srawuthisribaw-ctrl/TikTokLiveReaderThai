======================================================================
ประวัติการอัปเดตและรายการเปลี่ยนแปลงโปรแกรม TikTok Live Reader Thai
======================================================================

เวอร์ชัน: 3.0.1 (รุ่นปัจจุบัน)
----------------------------------------------------------------------
[1] รายการปรับปรุง:
- แยกคีย์ลัดเปิดหน้าจอตั้งค่าระบบหลักและเสียงอ่านให้ชัดเจนไม่สับสน:
  * ปุ่ม F9: เปิดตรงเข้าแท็บ "ตั้งค่าเสียงอ่านสังเคราะห์ (TTS)"
  * ปุ่ม F10: เปิดตรงเข้าแท็บ "ตั้งค่าระบบหลักและภาษา/การอ่านข้อมูล"
- แก้ไขข้อความป้ายกำกับปุ่มและเมนู (Labels) ในระบบภาษาไทยและภาษาอังกฤษให้ตรงกับหน้าต่างตั้งค่าจริง
- อัปเดตโครงสร้างระบบตรวจสอบและอัปเดตเวอร์ชันอัตโนมัติ (Auto-Updater) ให้เชื่อมโยงกับ GitHub Repository หลักของโครงการโดยตรง
- จัดระเบียบการเลือกตัวอ่านเสียงสังเคราะห์ (TTS Engines) ใหม่เป็น 4 หมวดหมู่หลัก (NVDA, JAWS, เสียงระบบ Windows, เสียงออนไลน์)
- พัฒนาระบบกรองรายชื่อเสียงพูดให้สัมพันธ์ตามหมวดหมู่ที่เลือก ป้องกันการเลือกตัวอ่านและรายชื่อเสียงสับสนหรือซ้ำซ้อน

เวอร์ชัน: 3.0.0
----------------------------------------------------------------------
[1] รายการปรับปรุง:
- พัฒนาระบบตรวจสอบและดาวน์โหลดการอัปเดตโปรแกรมอัตโนมัติ (Auto-Updater) ผ่านตัวประมวลผลแยก updater.exe ในเครื่องผู้ใช้
- แก้ไขปัญหาคีย์ลัด F6 ทับซ้อน และทำระบบสั่งหยุด/เงียบเสียงพูด (TTS) ทันที
- เพิ่มปุ่มคีย์ลัดด่วน Ctrl + Shift + G สำหรับสลับเปิด/ปิดการอ่านของขวัญ (Gift)
- เพิ่มและแก้ไขระบบสไตล์เสียงอ่านตลก (Robot, Child, Old, Fast, Slow, Funny) ให้รองรับการปรับแต่งความเร็วเสียงบน Microsoft OneCore (เช่น Pattara), Edge TTS และ Google TTS
- เพิ่มคู่มือแนะนำการขอและตั้งค่า Gemini API Key สำหรับผู้ใช้งานทั่วไปและผู้พิการทางสายตา (Screen Reader users) ในเอกสารคู่มืออย่างละเอียด

เวอร์ชัน: 2.5.1 และรุ่นก่อนหน้า
----------------------------------------------------------------------
- ระบบอ่านออกเสียงคอมเมนต์สด (Chat Comments)
- ระบบอ่านแจ้งเตือนการเข้าห้อง (Join), การกดถูกใจ (Like), การแชร์ไลฟ์ (Share) และการสมัครสมาชิก VIP
- ระบบบอร์ดปุ่มกดเอฟเฟกต์เสียงประกอบ (Sound Effect Board - Alt+F10 / F1-F9)
- ระบบสุ่มสถิติและรายงานผลแชทสด (F8) และอันดับสตรีมเมอร์ (F7)
- ระบบเครื่องเล่นเพลงคำขอผู้ชม (Ctrl+Shift+M) และเครื่องเล่นวิทยุออนไลน์ (Ctrl+Shift+R)
- รองรับการเข้าถึงแบบสมบูรณ์ 100% ด้วยคีย์บอร์ดและหน้าจออ่านออกเสียง (NVDA / JAWS / Narrator)
