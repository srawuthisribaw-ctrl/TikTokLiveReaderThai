----------------------------------------------------------------------
[1] แนะนำโปรแกรมเบื้องต้น
----------------------------------------------------------------------
โปรแกรม TikTok Live Reader เป็นผู้ช่วยไลฟ์สดสำหรับคนตาบอดและผู้พิการทางสายตา
ช่วยเชื่อมโยงข้อมูลเหตุการณ์ทั้งหมดในห้องไลฟ์สด เช่น แชทข้อความคอมเมนต์,
ของขวัญ, ยอดกดถูกใจ, ผู้สมัครสมาชิกช่อง, และการแชร์ไลฟ์สด
มาแปลงเป็นข้อมูลประมวลผลเสียงพูดออกลำโพง (Text-To-Speech) แบบเรียลไทม์
พร้อมทั้งมีระบบตอบคำถาม AI, เกมแชทสร้างปฏิสัมพันธ์ และระบบเปิดสถานีวิทยุประกอบ

----------------------------------------------------------------------
[2] ปุ่มลัดระบบและคีย์ลัดคีย์บอร์ด (Global Hotkeys)
----------------------------------------------------------------------
โปรแกรมถูกออกแบบให้ควบคุมผ่านแป้นพิมพ์ได้ 100% แม้ในขณะที่สตรีมเมอร์กำลังเล่นเกมส์
หรือย่อหน้าต่างโปรแกรมลง (Minimize) โดยกดปุ่มดังนี้:

• F1   : อ่านสถานะระบบปัจจุบัน (เช่น เชื่อมต่ออยู่กับห้องใด สัญญาณดีไหม)
• F2   : เริ่มการเชื่อมต่อห้องไลฟ์สด TikTok (พิมพ์ไอดีในกล่องสตรีมก่อนกด)
• F3   : หยุดการเชื่อมต่อห้องไลฟ์สดปัจจุบัน
• F4   : เปิด หรือ ปิด การอ่านแชทคอมเมนต์ของผู้ชม
• F5   : เปิด หรือ ปิด การอ่านรายงานเมื่อผู้ชมเข้าห้อง
• F6   : หยุดเสียงพูดนำทางปัจจุบันทันที (Mute) และล้างคิวเสียงสะสมที่หน่วงอยู่
• F7   : อ่านประกาศอันดับผู้ชมผู้สนับสนุน (Leaderboard) สูงสุด 3 อันดับแรก
• F8   : อ่านยอดสถิติสดในขณะนี้ (จำนวนคนดูรวม, ข้อความรวม, ยอดเพชรสะสม)
• F9   : เปิดหน้าต่างตั้งค่าระบบเสียงอ่านและการเข้าถึง (ตัวอ่าน/ความเร็ว/ความดัง)
• F10  : เปิดหน้าต่างตั้งค่าการทำงานหลักของโปรแกรม (และ Mixer Profile)
• F11  : บันทึกและส่งออกประวัติสตรีมสดปัจจุบันเป็นไฟล์ TXT, CSV, และ JSON
• F12  : อ่านรายงานสรุปไลฟ์สด (จำนวนแชทสะสม, มูลค่าเพชรรวม, คนติดตามเพิ่ม)

• Ctrl + Shift + A : เปิดกล่องถามคำตอบเอไอส่วนตัวสตรีมเมอร์ (Streamer AI)
• Ctrl + Shift + S : รายงานสรุปตัวเลขสถิติอย่างด่วน
• Ctrl + Shift + G : เปิด หรือ ปิด การอ่านรายงานของขวัญ (Gift alerts)

• Ctrl + P : เล่นเพลง หรือหยุดเล่นเพลงชั่วคราว (Pause/Resume)
• Ctrl + N : ข้ามไปเล่นเพลงถัดไปในเพลย์ลิสต์
• Ctrl + B : ย้อนกลับไปเล่นเพลงก่อนหน้า
• Ctrl + M : ปิดเสียงหรือเปิดเสียงเพลง (Music Mute Toggle)
• Ctrl + Shift + M : เปิดหน้าต่างจัดการเครื่องเล่นเพลงและคิวขอเพลงสะสม

• Alt + F1 ถึง Alt + F8 : เล่นเสียงเอฟเฟกต์ซาวด์บอร์ดด่วน (หัวเราะ, ปรบมือ, เชียร์, ว้าว, ตึ่งโป๊ะ, กลองม้วน, ชนะ, แพ้)
• Alt + F9 : สุ่มเล่นเสียงเอฟเฟกต์ตลกในระบบ
• Alt + F10 : เปิดหน้าต่างซาวด์บอร์ดเพื่อเล่นและทดลองฟังเสียงประกอบทั้งหมด 18 เสียง

----------------------------------------------------------------------
[3] เมนูแถบเครื่องมือภาษาไทยทั้งหมด
----------------------------------------------------------------------
ผู้ใช้งานสามารถกดปุ่ม Alt เพื่อเรียกเมนูบาร์ขึ้นมา และเลื่อนปุ่มลูกศรซ้ายขวาเพื่อเลือกเมนู:

1. ไฟล์: ใช้บันทึกประวัติการไลฟ์เป็นข้อความ (.txt) และไฟล์ Excel (.csv) หรือ ออกจากโปรแกรม
2. เชื่อมต่อห้อง: ควบคุมการเริ่มและหยุดการเชื่อมต่อ
3. สถิติสะสม: เรียกเปิดหน้าต่างสรุปและอ่านอันดับคนดูที่มีแต้มสูงสุด
4. ระบบเสียง: เปิดตั้งค่าเสียงพูดและคัดกรองคำหยาบ
5. การเข้าถึง:
   - สลับระบบนำทางคนตาบอด (Advanced Blind Mode)
   - สลับสีหน้าจอคอนทราสต์สูง (High Contrast) เหมาะสำหรับคนสายตาเลือนราง
   - สลับตัวอักษรขนาดใหญ่ (Large Font)
6. เครื่องมือและเกม: 
   - สั่งรันเกมสร้างปฏิสัมพันธ์ในไลฟ์สด
   - เปิด/ปิด/สลับ ช่องวิทยุออนไลน์
   - เครื่องเล่นเพลงสตรีมเมอร์ (Ctrl+Shift+M)
   - ซาวด์บอร์ดเอฟเฟกต์เสียงประกอบ (Alt+F10)
7. ช่วยเหลือ: เปิดอ่านคู่มือเวอร์ชันนี้และสืบค้นความช่วยเหลืออัปเดตเวอร์ชัน

----------------------------------------------------------------------
[4] ระบบคะแนน เลเวล และคำสั่งแชทสำหรับผู้ชม (Chat Commands)
----------------------------------------------------------------------
เมื่อผู้ชมเข้ามาในห้องไลฟ์สด จะสามารถส่งคำสั่งแชทด้านล่างนี้เพื่อโต้ตอบ:

• !ช่วยเหลือ : อ่านรายการคำสั่งทั้งหมดที่บอทรองรับ
• !คะแนน   : แสดงคะแนนสะสมและระดับเลเวล (เลเวล 1 ถึง 100)
• !อันดับ   : ประกาศรายชื่อผู้สนับสนุนระดับท็อป 3 อันดับแรกในแชท
• !ภารกิจ   : เช็คเควสประจำวันของตนเอง (เช่น พิมพ์ครบ 20 คอมเมนต์ ได้แต้มเพิ่ม)
• !หมุน     : หมุนวงล้อนำโชคแลกรางวัล (หัก 50 คะแนนเพื่อหมุน)
• !ร้านค้า   : แสดงรายการไอเทมในช็อปแต้มสะสม
• !ซื้อ      : แลกซื้อไอเทมในร้านค้าแต้ม เช่น !ซื้อ ขอเพลง
• !เพลง     : ส่งคำขอเพลงเข้าสู่คิวสตรีมเมอร์ เช่น !เพลง ขอใจเธอแลกเบอร์โทร
• !ถาม หรือ @ตามด้วยคำถาม : ถามคำถามปัญญาประดิษฐ์ (AI) บอทจะตอบเสียงนำทางอัตโนมัติ (เช่น @วันนี้ฝนตกไหม หรือ @ วันนี้ฝนตกไหม)

----------------------------------------------------------------------
[5] มิกเซอร์เสียงแยกช่อง และสไตล์เสียงอ่านตลก (Funny TTS)
----------------------------------------------------------------------
• เพื่อความเพลิดเพลินสูงสุดขณะสตรีม โปรแกรมรองรับการปรับระดับเสียงมิกเซอร์ 6 ช่อง:
  1. เครื่องเล่นเพลง (Music)
  2. ข้อความคอมเมนต์ (Comment)
  3. แจ้งเตือนของขวัญ (Gift)
  4. เสียงเอฟเฟกต์ซาวด์บอร์ด (SFX)
  5. เสียงนำทางหลัก (TTS)
  6. เสียงคำถามตอบ AI (AI)
• สามารถบันทึกโปรไฟล์เสียงได้ 4 โหมด: โหมดไลฟ์ปกติ (Normal), โหมดเล่นเกม (Game), โหมดร้องเพลง (Singing), โหมดกิจกรรมพิเศษ (Special)
• รองรับเสียงตลกสังเคราะห์ (Funny TTS Style) ถึง 7 รูปแบบในแท็บเสียงอ่าน TTS:
  ปกติ (Normal), หุ่นยนต์ (Robot), เด็ก (Child), คนแก่ (Old), พูดเร็ว (Fast), พูดช้า (Slow), ตลกสูง (Funny)
  (สำหรับเอนจิน SAPI5, Windows OneCore และ Edge TTS)

----------------------------------------------------------------------
[6] การคัดกรองสแปมและคิวความสำคัญเสียงพูด (Priority Audio Queue)
----------------------------------------------------------------------
• เพื่อป้องกันปัญหาเสียงแชทตีกันในห้องไลฟ์สดขนาดใหญ่ ระบบจะเรียงคิวอัตโนมัติ:
  1. ของขวัญและโดเนทชิ้นใหญ่ (ระดับความสำคัญ 10) - จะถูกอ่านก่อนเสมอ
  2. ผู้กดติดตามใหม่และ VIP (ระดับความสำคัญ 8)
  3. คอมเมนต์แชททั่วไป (ระดับความสำคัญ 5)
  4. ผู้ชมที่เดินเข้าห้องปกติ (ระดับความสำคัญ 2)
• ระบบมีการดักจับข้อความสแปม (Spam Control) หากผู้ชมพิมพ์ข้อความซ้ำกันรวดเร็วเกินไป
  ระบบจะละเว้นการอ่านเพื่อไม่ให้เสียงนำทางหน่วง
• สตรีมเมอร์สามารถใส่ API Key ของโมเดล Google Gemini ได้ที่หน้าตั้งค่า เพื่อเพิ่มความสามารถ
  ให้ AI โต้ตอบฉลาดกับผู้ชมสดในแชท

----------------------------------------------------------------------
[7] ขั้นตอนการขอและตั้งค่า Gemini API Key สำหรับคนตาบอดอย่างละเอียด
----------------------------------------------------------------------
สำหรับคนตาบอดที่ต้องการเปิดใช้งานระบบผู้ช่วย AI โต้ตอบสดอัจฉริยะ สามารถทำการขอ API Key ได้ฟรีผ่านโปรแกรมอ่านหน้าจอ (NVDA หรือ JAWS) ตามขั้นตอนอย่างละเอียดดังนี้ค่ะ:

1. การเตรียมตัวก่อนดำเนินการ:
   - แนะนำให้ใช้เว็บเบราว์เซอร์ Google Chrome หรือ Microsoft Edge บนคอมพิวเตอร์ระบบปฏิบัติการ Windows
   - ทำการเข้าสู่ระบบ (Sign In) บัญชี Google (Gmail) บนเว็บเบราว์เซอร์ที่ใช้งานอยู่ให้เรียบร้อยก่อน เพื่อลดขั้นตอนการเข้าสู่ระบบที่ซับซ้อน

2. ขั้นตอนการเข้าสู่ระบบ Google AI Studio:
   - พิมพ์ลิงก์หรือเปิดหน้าเว็บ https://aistudio.google.com/ จากนั้นกดปุ่ม Enter
   - เมื่อหน้าเว็บโหลดเสร็จสิ้น ในกรณีที่เป็นการเข้าใช้งานครั้งแรก ระบบจะแสดงหน้าเงื่อนไขข้อตกลงการใช้บริการ (Terms of Service)
   - ให้กดปุ่ม Tab เพื่อเลื่อนหา "ช่องเครื่องหมายยอมรับข้อตกลง" (หรือใช้คีย์ลัด x ใน NVDA เพื่อหา Checkbox)
   - เมื่อได้ยินเสียงโปรแกรมอ่านหน้าจอพูดว่า "I agree to the terms... checkbox not checked" ให้กดปุ่ม Spacebar 1 ครั้ง เพื่อทำเครื่องหมายเลือก (Checked)
   - จากนั้นกดปุ่ม Tab ไปเรื่อยๆ จนเจอปุ่ม "Continue" หรือ "Accept" (โปรแกรมจะพูดว่า "Continue button") แล้วกด Enter เพื่อผ่านเข้าไปยังหน้าหลัก

3. ขั้นตอนการกดสร้างคีย์ API (Create API Key):
   - ในหน้าหลักของ Google AI Studio ให้กดคีย์ลัดตัวอักษร H เพื่อเลื่อนตามหัวข้อใหญ่ (Headings) หรือกดปุ่ม Tab ไปเรื่อยๆ จนกว่าจะได้ยินเสียงอ่านว่า "Get API Key button" (ปุ่มรับ API Key) หรือ "Create API Key"
   - เมื่อพบแล้วให้กดปุ่ม Enter หรือ Spacebar
   - หน้าจอจะสลับเข้าสู่ส่วนจัดการคีย์ ให้กดปุ่ม Tab เพื่อหาปุ่มที่อ่านว่า "Create API Key in new project" (สร้างคีย์ในโปรเจกต์ใหม่) หรือ "Create API Key"
   - กดปุ่ม Enter เพื่อยืนยันคำสั่ง ระบบจะขึ้นสถานะกำลังสร้างคีย์ ( Generating ) เป็นเวลา 2-3 วินาที

4. ขั้นตอนการคัดลอก (Copy) รหัสคีย์:
   - เมื่อสร้างเสร็จ ระบบจะแสดงกล่องข้อความป๊อปอัป (Dialog Box) เด้งขึ้นมาตรงหน้าจอ
   - ให้กดปุ่ม Tab ไปเรื่อยๆ จนได้ยินเสียงโปรแกรมอ่านหน้าจอพูดว่า "Copy button" (ปุ่มคัดลอก) หรือ "Copy API Key button"
   - กดปุ่ม Enter หรือ Spacebar 1 ครั้ง เพื่อคัดลอกรหัสคีย์เก็บไว้ในหน่วยความจำชั่วคราว (Clipboard) ของคอมพิวเตอร์
   - จากนั้นกดปุ่ม Escape 1 ครั้ง เพื่อปิดกล่องป๊อปอัปนั้นไป

5. ขั้นตอนการนำคีย์ไปตั้งค่าในโปรแกรม TikTok Live Reader:
   - สลับหน้าต่างกลับมาที่โปรแกรม TikTok Live Reader
   - กดปุ่มลัด F9 (หน้าต่างตั้งค่าระบบเสียงอ่านและการเข้าถึง)
   - กดปุ่ม Tab เพื่อเลื่อนโฟกัสไปเรื่อยๆ จนกว่าจะได้ยินโปรแกรมอ่านออกเสียงว่า "ช่องป้อนรหัส Gemini API Key" หรือ "Gemini API Key edit text"
   - กดปุ่ม Ctrl + V เพื่อวางรหัส API Key ที่คัดลอกมาลงในช่อง
   - กดปุ่ม Tab เลื่อนไปยังปุ่ม "บันทึกการตั้งค่า" (หรือ Save Settings button) จากนั้นกดปุ่ม Enter หรือ Spacebar เพื่อสิ้นสุดการตั้งค่า
   *ระบบ AI ทั้งเสียงวิเคราะห์ส่วนตัว (Ctrl+Shift+A) และบอทตอบคอมเมนต์อัตโนมัติ (!ถาม) จะพร้อมทำงานในโหมดออนไลน์ทันทีค่ะ*
